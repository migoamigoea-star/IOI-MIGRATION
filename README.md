# IOI Migration Project

This repository contains the source documentation for the **IOI Migration Project** — a structured initiative to migrate business processes and data to **Microsoft Dynamics 365 Business Central** using the AL (Application Language) development platform.

## Overview

The IOI Migration Project covers the full lifecycle of planning, developing, testing, and deploying a Business Central solution. This repository serves as the central knowledge base for all project documentation, including architecture decisions, migration plans, setup guides, and changelogs.

## Project Goals

- Migrate existing business workflows and data to Dynamics 365 Business Central
- Develop custom AL extensions to meet business requirements
- Ensure data integrity and continuity throughout the migration process
- Provide clear documentation for all stakeholders and developers
